# 🎯 프리코스 오픈 미션 과제 - 격자 야구 게임

## 🚀 기능 구현 목록
> - 내용이 추가 또는 수정될 수 있다.
> - 기능 작동 순서대로 작성한다.
> - 기능을 구현했다면 해당 기능 목록에 체크한다.

### 0. 사용자에게 두 가지의 게임 모드를 보여준다.
- [x] '혼자 플레이' 모드 버튼을 클릭하면 
- [x] '컴퓨터와 대결' 모드

### 1. 사용자에게 게임 화면을 보여준다.
- [x] 게임 규칙 패널, 게임 타이틀 패널, 게임 진행 요소 패널, 격자 패널 총 4개의 패널로 구성해 보여준다.
- [x] 사용자가 격자 크기를 입력한다.

### 1-1. 격자 크기를 변경한다.
- [x] 격자 크기 입력값을 검증한다.
  - [x] 입력값이 숫자 형태가 아니면 사용자에게 에러메시지를 보여준다.
  - [x] 입력값이 정해진 범위가 아니면 사용자에게 에러메시지를 보여준다.
- [x] 격자 크기 입력값을 정수로 변환한다.
- [x] 격자 크기 입력값이 정상적으로 입력되면 격자 크기를 변경하고 게임 내용을 초기화한다.
  - [x] 정상 입력되었음을 알리는 메시지를 출력한다.

### 2. 아웃 칸 3개를 결정한다.
- [x] 아웃 칸 3개를 랜덤으로 결정한다.

### 3. 격자 칸을 선택했을 때 나오는 반응을 구현한다.
- [x] 아웃 칸을 선택하면 아웃임을 알린다.
- [x] 그 외의 칸을 선택하면 선택한 칸이 N Strike M Ball 임을 알린다.
- [x] 현재 투구수와 아웃 카운트를 갱신한다.

### 4. 아웃카운트가 3이 되면 투구수를 표시하고 게임을 종료한다.
- [x] 아웃카운트가 3이 되면 우승 메시지와 투구수를 표시하고 게임을 종료한다.
